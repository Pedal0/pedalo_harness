# Project memory (MANDATORY WORKFLOW)

The .pedalo/ folder is your persistent memory of this project:
MAP.md (structure), lessons.md (past problems→solutions),
decisions.md (choices made), state.md (current task).

## At the START of any task
Read .pedalo/MAP.md before doing anything else, to understand the project structure. If a file you need is already described there, use that description instead of re-reading the whole file.

## BEFORE giving your final answer
This step is not optional. Check both points:
1. For each file you read during THIS task that has no description yet in MAP.md: use edit_file on .pedalo/MAP.md to append ' — <one-line description>' to that file's line.
2. If during this task an approach failed and another succeeded (wrong tool name, wrong path, error then fix): use edit_file on .pedalo/lessons.md to append one line, e.g. '- edit_file: old_str matching a name that appears twice fails → include the surrounding block to make it unique'.
